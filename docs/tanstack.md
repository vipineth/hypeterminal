# TanStack

> TanStack provides high-quality, open-source libraries for web development including routing, data fetching, state management, tables, forms, and more. All libraries are framework-agnostic with first-class support for React, Vue, Solid, Svelte, and Angular.

## Documentation

### Data and State Management

- [TanStack Start](https://tanstack.com/start/latest/docs/framework/react/overview): Full-stack Framework powered by TanStack Router for React and Solid
- [TanStack Router](https://tanstack.com/router/latest/docs/framework/react/overview): Type-safe Routing for React and Solid applications
- [TanStack Query](https://tanstack.com/query/latest/docs/framework/react/overview): Powerful asynchronous state management, server-state utilities and data fetching
- [TanStack DB](https://tanstack.com/db/latest/docs/overview): The reactive client-first store for your API
- [TanStack Store](https://tanstack.com/store/latest/docs/overview): Framework agnostic data store with reactive framework adapters
- [TanStack AI](https://tanstack.com/ai/latest/docs/getting-started/overview): A powerful, open-source AI SDK with a unified interface across multiple providers

### Headless UI

- [TanStack Table](https://tanstack.com/table/latest/docs/introduction): Headless UI for building powerful tables & datagrids
- [TanStack Form](https://tanstack.com/form): Headless UI for building performant and type-safe forms

### Performance

- [TanStack Virtual](https://tanstack.com/virtual/latest/docs/introduction): Headless UI for Virtualizing Large Element Lists
- [TanStack Pacer](https://tanstack.com/pacer/latest/docs/overview): Framework agnostic debouncing, throttling, rate limiting, queuing, and batching utilities

### Tooling

- [TanStack Devtools](https://tanstack.com/devtools): Centralized devtools panel for TanStack libraries and other custom devtools
- [TanStack Config](https://tanstack.com/config): Configuration and tools for publishing and maintaining high-quality JavaScript packages

A powerful React router for client-side and full-stack react applications. Fully type-safe APIs, first-class search-params for managing state in the URL and seamless integration with the existing React ecosystem.

Powerful asynchronous state management, server-state utilities and data fetching. Fetch, cache, update, and wrangle all forms of async data in your TS/JS, React, Vue, Solid, Svelte & Angular applications all without touching any "global state"
